# Smart AI Text Summarizer

This project is an AI-powered text summarizer built using:

* Python
* Flask
* NLP
* Scikit-learn
* spaCy
* FuzzyWuzzy

## Features

* Summarizes any text
* TF-IDF based sentence ranking
* Cosine similarity
* Fuzzy similarity score
* Beautiful UI using HTML & CSS

## Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

```plaintext
http://127.0.0.1:5000
```
